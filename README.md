I. INTRODUCTION
Recruitment is a critical function in human resource management, yet it remains highly manual and time-consuming. Organizations often receive hundreds of resumes for a single job opening, forcing recruiters to spend excessive hours on initial screening. This process not only slows down hiring but also introduces human bias and fatigue-related errors.

With advancements in Artificial Intelligence (AI) and Natural Language Processing (NLP), automated screening systems can significantly improve recruitment efficiency. This project aims to design and implement an AI-powered resume shortlisting system that objectively evaluates candidates based on predefined job requirements and produces structured, actionable results.

II. PROBLEM STATEMENT
Recruiters face a Resume Overload problem, spending an average of 23 hours per job role on resume screening. Studies indicate that approximately 88% of applicants do not meet basic eligibility criteria. Manual evaluation increases time-to-hire and operational cost and increases the risk of overlooking qualified candidates.

III. OBJECTIVES
• Automate resume extraction and shortlisting  
• Reduce recruiter workload  
• Provide objective evaluation  
• Rank candidates using weighted scoring  
• Improve recruitment efficiency  

IV. RELATED WORK
Previous recruitment automation systems relied on keyword-based filtering. Modern LLM-based systems provide semantic understanding, reducing false rejections.

V. SYSTEM ARCHITECTURE
The system follows a three-layer architecture:
Ingestion Layer – Google Drive trigger
Processing Layer – OCR and AI scoring
Logging Layer – Google Sheets output
